<h2 id="math-formula-1">다각형의 면적 공식</h2>

[참조](https://ko.wikihow.com/%EB%8B%A4%EA%B0%81%ED%98%95-%EB%84%93%EC%9D%B4-%EA%B5%AC%ED%95%98%EA%B8%B0)

예시) 오각형

- A (x1, y1)
- B (x2, y2)
- C (x3, y3)
- D (x4, y4)
- E (x5, y5)

`Z1 = x1 * y2 + x2 * y3 + ... + x4 * y5 + x5 * y1`

`Z2 = x2 * y1 + x3 * y2 + ... + x5 * y4 + x1 * y5`

오각형의 면적 : `(Z1 - Z2) / 2`
